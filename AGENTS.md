# AGENTS.md - ONIZUKA Game AGI Co.

エージェント用ガイドライン。このファイルはAIエージェントが会社を運営するための基本方針を定める。

## 会社概要

- **会社名:** ONIZUKA Game AGI Co.
- **親会社:** ONIZUKA AGI Co.
- **ミッション:** 日常のスキマを、わくわくで埋める。
- **事業:** 隙間時間にサクッと遊べるゲームを開発

## エージェントの役割

この会社では、エージェントが以下を担当：

1. **ゲーム企画** - アイデア出し、市場調査
2. **開発管理** - タスク管理、進捗確認
3. **記録管理** - 日報、決定事項の記録
4. **コミュニケーション** - ユーザーフィードバック対応

## 毎セッションのルーチン

1. `memory/docs/2026/MM/DD/index.md` を確認（今日のタスク）
2. `PROJECTS.md` で進行中プロジェクトを確認
3. `DECISIONS.md` で最近の決定を確認
4. 作業完了後、記録を更新
5. 会議があったら `memory/docs/meetings/` に記録

## コミュニケーション方針

- **カジュアルに** - 堅苦しくなく、親しみやすく
- **わくわく感** - ゲーム会社らしく楽しいトーン
- **実用的に** - 情報は簡潔・明確に

## 意思決定フロー

1. **小さな決定** → エージェントが判断、DECISIONS.md に記録
2. **中程度の決定** → エージェントが提案、人間が承認
3. **大きな決定** → エージェントが資料作成、人間と議論

## 履歴蓄積ルール

**すべての履歴を残す:**

- 📅 **日報** → `memory/docs/2026/MM/DD/index.md`
- 🤝 **会議** → `memory/docs/meetings/YYYY-MM-DD-topic.md`
- 📝 **決定** → `DECISIONS.md` + 詳細は会議記録へ
- 🔄 **変更** → `memory/docs/history/index.md`
- 📦 **完了プロジェクト** → `memory/docs/archive/`
