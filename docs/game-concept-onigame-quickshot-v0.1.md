# onigame-quickshot v0.1

## 目的

短時間で繰り返し遊べる、1画面完結の回避アクションを作る。  
「30秒で理解できて、1分で再挑戦したくなる」体験を最小ゴールにする。

## 前提制約

- GitHub Pages 完全静的配信
- 外部API不要
- ブラウザ内完結
- 実装は小さく分割し、数時間単位で前進できること

## コアループ

1. 1クリックで開始
2. 障害物を回避して生存時間を伸ばす
3. ミスでゲームオーバー
4. スコア表示後、即リトライ

## MVP仕様

- 1画面のみ（シーン遷移なし）
- プレイヤー移動: 矢印キー or WASD
- 障害物: 画面外から流入、速度は時間で上昇
- 終了条件: 1ヒットで終了
- スコア: 生存秒数
- 保存: `localStorage` にベストスコアのみ保存

## 難易度設計（初期値）

- 0-20秒: 低密度
- 21-40秒: 中密度
- 41-60秒: 高密度
- 60秒以降: 速度上昇のみ継続

## UI要件

- タイトル
- 現在スコア
- ベストスコア
- Start / Retry ボタン
- モバイルでも最低限見切れないレイアウト

## 非目標（v0.1ではやらない）

- バックエンド連携
- マルチプレイ
- AGI対戦要素
- 複雑な演出システム

## 受け入れ基準

- 初見で 30 秒以内に遊び始められる
- 1プレイが 3 分以内に収まる
- GitHub Pages でそのまま動作する
- ベストスコア保存が動作する
