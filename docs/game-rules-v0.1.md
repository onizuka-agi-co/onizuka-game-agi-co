# Grid Tactics ルール仕様 v0.1

- 作成日: 2026-03-09
- 目的: 実装ブロッカーの解消（MVP向け最小仕様の固定）
- 対象: プレイヤー vs AGI（ブラウザ内完結）

## 1. ゲーム概要

- 盤面: 7x7 グリッド
- 陣営: プレイヤー2体 / AGI2体
- ユニット種別: Striker, Guardian
- 勝利条件:
  - 相手ユニットを全滅させる
  - 15ターン終了時は残HP合計が高い側を勝者とする（同値は引き分け）

## 2. ターン進行

1. プレイヤーターン開始
2. プレイヤーが各ユニットを1回ずつ行動
3. AGIターン開始
4. AGIが各ユニットを1回ずつ行動
5. ターンカウント +1
6. 勝敗判定（決着なら終了、未決着なら1へ）

- 1ターン = プレイヤー行動フェーズ + AGI行動フェーズ
- ターン上限: 15

## 3. ユニット仕様（MVP）

### Striker
- HP: 10
- 移動力: 3マス
- 射程: 1-2マス
- 攻撃力: 4

### Guardian
- HP: 14
- 移動力: 2マス
- 射程: 1マス
- 攻撃力: 3

## 4. 行動ルール

- 各ユニットは自身のフェーズ中に1回行動できる
- 行動は「移動のみ」または「攻撃のみ」または「移動 + 攻撃」のいずれか
- 移動:
  - マンハッタン距離で移動力以内
  - 盤面外・占有マスへの移動不可
- 攻撃:
  - 射程内の敵1体を対象
  - ダメージ計算: 基礎攻撃力 - カバー補正（最低1）

## 5. カバー仕様

- 盤面に固定のカバータイルを 3 箇所配置
- カバー上のユニットは被ダメージ -1
- カバー補正は攻撃ごとに適用

## 6. AGI（Easy）

- 優先順位:
  1. 攻撃可能なら攻撃（最もHPが低い敵を優先）
  2. 攻撃不能なら最も近い敵へ接近
- 不正手（盤面外、占有、射程外）は選ばない
- 思考時間目標: 1手あたり 3 秒以内

## 7. UI表示（最低限）

- 現在ターン数
- 現在フェーズ（Player / AGI）
- ユニットHP
- 移動可能範囲ハイライト（緑）
- 攻撃可能範囲ハイライト（赤）
- カバータイル表示

## 8. 非対象（v0.1では実装しない）

- サウンド
- 複数マップ
- ユニットカスタマイズ
- 対戦履歴
- マルチプレイ

## 9. 受け入れ基準（v0.1）

- 1試合を開始から終了まで完走できる
- ターン不整合が発生しない
- AGIが有効手のみを選択する
- 15ターン上限で必ず試合が終了する
