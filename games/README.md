# Games Workspace

`games/` is the local workspace for game projects.

## Operating Rule

- `onigame-*` directories under this folder are independent Git repositories
- each game repository is expected to have its own GitHub Pages deployment
- the company repository keeps planning, decisions, and links, but does not track those game repos directly

## Current Local Repos

- `onigame-omikuji` - sample static omikuji app

## Notes

- `grid-tactics` remains as legacy in-repo code from before the per-game repo rule was adopted
- future game repos should prefer the `onigame-<name>` naming convention
