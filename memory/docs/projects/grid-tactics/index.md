# Grid Tactics

## 概要
7x7グリッドのターン制タクティカルゲームMVP（プレイヤー vs AGI）。

## ステータス

- **開始日:** 2026-03-08
- **現在:** active
- **担当:** エージェントチーム（Product Owner, Tech Lead, Designer, QA）

## 目標
5-10分で1試合完走できる最小プロトタイプを早期に提供する。

## 仕様

### ゲーム概要
- プラットフォーム: Web（GitHub Pages）
- プレイ時間: 5-10分
- ターゲット: 隙間時間で短く遊びたいプレイヤー

### 操作
- クリック/タップでユニット選択
- 移動先指定
- 攻撃対象指定

### 画面
- 7x7グリッド
- ユニット表示（HP）
- ターン/フェーズ表示
