# Start Checklist

Use this checklist before declaring a new ONIGAME repo ready.

## Local repo

- repo folder exists under `games/onigame-<slug>/`
- `git status --short` is clean
- `README.md` has a clear hero, badges, and practical usage sections
- `README.ja.md` exists when the repo is aimed at ONIZUKA's normal Japanese-speaking audience
- at least one reusable SVG mark exists under `assets/`
- static entry files exist and work together

## Remote repo

- remote repo exists under `onizuka-agi-co`
- `origin` is configured
- `main` is pushed
- repo description is set
- repo homepage is set
- repo topics are set

## Pages readiness

- homepage points to the expected Pages URL
- app is deployable from static files only
- Pages enablement was attempted
- if Pages was blocked, the exact blocker is recorded

## Company workspace

- child repo contents were not staged in the company repo
- any requested company logs or indexes were updated
- the final response includes the repo URL and current publish state
